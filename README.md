# cc-clone-mintlify

# Mintlify Landing Page Clone (HTML + CSS)

## Project Overview

This Project is clone of Mintlify using HTML and CSS only.
Landing page design including:
-Navigation Bar

- Announcement Banner
- Hero section
- Feature grid layout
- AI documentation sections
- Text overlay on images
- Responsive card design
  This project was created for **learning layout, CSS grid, flexbox, and UI design concepts**.

## Features

- Responsive grid layout
- Feature cards with modern styling
- Image overlay text section
- Documentation UI mockup design
- Simple and clean CSS structure

## Technologies Used

- HTML5
- CSS3 (Grid + Flexbox)

## Project Structure

```
project-folder/
│
├── index.html
├── style.css
├── images/
├── Screenshot/
└── README.md
```

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone [<your-repo-url>](https://github.com/PANKAJ6181/cc-clone-mintlify)
   ```
2. Open the folder.
3. Double-click `index.html`
   OR open in browser using Live Server.

## Learning Goals

- CSS Grid & Flexbox layouts
- Modern SaaS UI design
- Overlay text on images
- Clean responsive styling
- Frontend structure understanding
